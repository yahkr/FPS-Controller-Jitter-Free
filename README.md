# FPS-Jitter-Free
 
This project provides a jitter free fps controller that works with the Godot 4.4 physics interpolation. 

```
Controls:
player_forward (W)
player_backward (S)
player_left (A)
player_right (D)
quit (Escape or Alt+Q)
```
